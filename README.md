# awesome vhdl

VHDL support for Visual Studio Code

## Demo

![gif1](https://github.com/puorc/awesome-vhdl/raw/master/images/demo.gif)

## Features
- syntax highlighting.
- code snippets.
- code completion.
- brace matching.
- line and block commenting.
- other features and better syntax analysis will be added soon.
